# ERP Architecture Learning Project

This project is an attempt to understand ERP architecture from scratch by building and deploying a full-stack application on an Amazon Web Services Amazon EC2 instance.

The primary focus of this project is the operational side of application management — including deployment, server configuration, hosting, monitoring, maintenance, and troubleshooting in a production-like environment.

The application itself is largely “vibe coded,” as the main objective is not application development, but gaining practical exposure to infrastructure operations, debugging, and understanding how different components of the stack work together in a live environment.
